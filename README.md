## About

I'm an ECE student focused on **digital system design, RTL development, and VLSI/CMOS technology**. I design and verify combinational and sequential circuits in Verilog and SystemVerilog, and I'm currently extending that into embedded systems and FPGA-based implementation.

- 🔭 **Currently building** — logic gates, ALUs, encoders/decoders, sequential circuits
- 🌱 **Currently learning** — FPGA implementation, embedded systems, and how software (C/Python) interfaces with hardware
- 💬 **Ask me about** — digital logic design, combinational/sequential circuits, HDL simulation
- 📫 **Reach me at** — [ayushsikarwar42@gmail.com](mailto:ayushsikarwar42@gmail.com)
